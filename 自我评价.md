# 2025/4/3

```sql
select instructor.salary from instructor
where dept_name = 'Statistics' limit 1;
```
用`limit`强行限制，会强制限制查询结果只返回**第一条记录**（即查询结果的第一行）

>如果没有 `ORDER BY`，数据库会以**不确定的顺序**返回这些记录（通常是物理存储顺序或索引顺序）
